Digital Publishing WAI-ARIA Module 1.1

DPUB-AAM is a web accessibility standard created by the W3C. It helps assistive technologies such as screen readers understand digital publications like ebooks, online textbooks, journals, and articles.

Main Goals

  • Improve accessibility for digital publishing
  • Help screen readers interpret content correctly
  • Create better navigation for users with disabilities
  • Support semantic structure in online publications
